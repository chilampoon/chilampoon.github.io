    Tools for rhythmicity detection

    Chi Lam Poon
    2019-12-31
    Circadian rhythm, RNA-seq

    Introduction

    In an early period, some tools can only detect rhythmicity, i.e. to tell you whether this gene or transcript or protein had cycling expression, and they cannot tell you that whether the features you tested had different amplitudes, phases, etc. in different conditions. Thus, some tools were later created for doing that and they called this process “differential rhythmicity“.

    There are two main types of methods, parametric and nonparametric. The difference between these two is just whether based on a known mathematical model or not.

    Parametric methods are able to detect weaker rhythms than nonparametric methods. However, parametric methods rely on stricter assumptions: often, measurement errors must have Gaussian distributions, and the methods are sensible to outliers.

    Nonparametric methods sacrifice some degree of power in order to gain robustness against outliers, and they make fewer assumptions about measurement errors.

    Summary

    Rhythmicity Detection

    Type Method Year Notes Others
    Parametric Fisher’s G test 2004 Exact test with g-statistics Fourier analysis
    Sinusoidal regression 2013 Curve fitting
    Cosinor regression 2014 Curve fitting
    Nonparametric JTK_CYCLE 2010 JT test + Kendall's tau correlations Symmetric rhythms only
    RAIN 2014 Extend JTK to include asymmetric rhythms

    Differential Rhythmicity

    Type Method Year Notes Others
    Parametric DODR 2016 Detects changes in periodic paramters 2 conditions only
    Nonparametric LimoRhyde 2018 Combination of cosinor regression and limma Allows multiple experimental variables
